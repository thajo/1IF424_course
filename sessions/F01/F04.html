Saker att göra med en array

I PHP finns det ett antal vanligt förekommande uppgifter/metoder 

$tal = 123;

// Att skapa en ny, tom array
$array = array(); 

// Att lägga något sist i arrayen
array_push($array, $tal); // läggs sist 
array_push($array, 12); // läggs sist 

// Hur många element/värden finns det i arrayen
$nr_of_elements = sizeof($array); 
echo $nr_of_elements; // Skriver ut 2

Multidimensional Arrays (lite överkurs)

Ibland behöver man beskriva lite mer komplex data

Produkt Pris Antal
Bultbtäda 499 12
Sprakcykel 876 2
Modellera 39.90 124 

$shop = array( 
  array("Bultbräda", 499, 12),
  array("Sparkcykel", 876 , 2),
  array("Modellera", 39.90 , 124) 
);
echo $shop[0][0]." costs ".$shop[0][1]." and you get ".$shop[0][2];
